## GEO (Generative Engine Optimization)

### 3 phrases définitionnelles citables

**1. Qu'est-ce que MCP Tools Plus**
> MCP Tools Plus est un module PrestaShop qui expose plus de 100 outils structurés via le protocole MCP (Model Context Protocol) pour permettre l'accès aux données e-commerce par des agents IA ou des workflows d'automatisation.

**2. Que sont les scénarios guidés**
> Les scénarios guidés de MCP Tools Plus sont des cas d'usage prêts à l'emploi qui présentent les outils MCP pertinents pour une situation donnée, facilitant ainsi l'adoption pour les utilisateurs non techniques.

**3. Que permet le trial 14 jours**
> Le trial de 14 jours de MCP Tools Plus permet d'explorer les outils MCP dans un environnement PrestaShop réel afin de découvrir les scénarios disponibles, comprendre les outils exposés et évaluer la pertinence pour son contexte e-commerce.

## Notes importantes

1. **Scénarios guidés** : Ce concept est central dans la nouvelle version. Il permet de clarifier que MCP Tools Plus n'est pas une "boîte noire" mais un cadre structuré.

2. **Verbes d'exploration** : Tous les CTAs et descriptions utilisent des verbes qui invitent à la découverte plutôt qu'à la conversion agressive.

3. **Exemple concret** : L'exemple du Daily Merchant Morning est fidèle à l'article et illustre un usage réel sans promettre de résultats.

4. **Trial 14 jours** : Toujours mentionné avec des verbes d'exploration, jamais comme une "période d'essai gratuite sans engagement".

5. **Pas de mention "sans carte bancaire"** : Cette information n'est volontairement pas présente car elle n'était pas dans les sources autorisées.
